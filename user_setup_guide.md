# FX Trade Insight 初回セットアップ手順書

アプリのインストール後、最初に行う設定手順を説明します。

---

## 1. ライセンスキーの入力

1. アプリを起動する
2. ライセンス認証画面が表示されるので、購入後に届いたメールに記載された **ライセンスキー** を入力する
   - 形式: `FTI-XXXX-XXXX-XXXX`
3. 「認証する」をクリックする
4. 「認証成功」と表示されたらメイン画面に進む

> ライセンス認証にはインターネット接続が必要です。認証できない場合は「よくある質問（FAQ）」の「ライセンス認証」セクションをご確認ください。

---

## 2. 動画保存フォルダの設定

録画した動画の保存先フォルダを指定します。

1. 左メニューの「設定・データ管理」を開く
2. 「録画・動画再生設定」セクションを開く
3. 「動画保存先」の「変更」ボタンをクリックし、保存したいフォルダを選択する
   - 推奨: 空き容量に余裕のあるドライブ（動画は数GB単位になる場合があります）
4. 「保存」をクリックして設定を確定する

---

## 3. CSV 取込フォルダの設定

FX 口座の取引履歴 CSV を読み込むためのフォルダを設定します。

1. 「設定・データ管理」→「CSV 取込設定」を開く
2. 「CSV 取込フォルダ」の「変更」ボタンをクリックし、フォルダを選択する
3. 選択したフォルダ内に、FX 口座ごとのサブフォルダを作成する

**フォルダ構成の例:**

```
CSV取込フォルダ/
├── みんなのFX/
├── LIGHT FX/
├── GMOクリック証券/
├── 楽天 FX/
└── OANDA証券/
```

> フォルダ名は「設定・データ管理」→「CSV 取込ガイド」に表示される口座名と一致させてください。

取引履歴 CSV のエクスポート方法は、アプリ内の「CSV 取込」画面にある「?」アイコン（取込ガイド）を参照してください。

---

## 4. MT5 EA 導入（画面録画との連携に必要）

FX Trade Insight の画面録画と MT5 の取引を自動リンクするには、MT5 に専用の EA（エキスパートアドバイザー）を導入します。

### 4-1. EA ファイルの入手

アプリのホーム画面にある「TradeHistoryViewer.ex5 をダウンロード」ボタンをクリックしてダウンロードしてください。

### 4-2. MT5 への EA 配置

1. MT5 を起動する
2. メニューバーの「ファイル」→「データフォルダを開く」をクリックする
3. 開いたフォルダ内の `MQL5 > Experts` フォルダに `TradeHistoryViewer.ex5` をコピーする
4. MT5 の「ナビゲーター」パネルを右クリック →「更新」をクリックする
5. 「エキスパートアドバイザー」一覧に `TradeHistoryViewer` が表示されることを確認する

### 4-3. MT5 で WebRequest を許可する

EA がアプリと通信するために、MT5 の WebRequest を許可する必要があります。

1. MT5 のメニューバーの「ツール」→「オプション」を開く
2. 「エキスパートアドバイザー」タブを開く
3. 「以下の URL の WebRequest を許可する」にチェックを入れる
4. URL 入力欄に `http://127.0.0.1` を追加する
5. 「OK」をクリックする

### 4-4. チャートへの EA アタッチ

1. MT5 で録画対象のチャートを開く
2. ナビゲーターから `TradeHistoryViewer` をチャートにドラッグ＆ドロップする
3. 設定画面で「自動売買を許可する」にチェックを入れ「OK」をクリックする
4. チャート右上に EA 名と笑顔アイコンが表示されれば完了

> チャート右上が笑顔アイコンにならない場合は、MT5 ツールバーの「自動売買」ボタン（または F7 キー）がOFFになっている可能性があります。ONにしてください。

---

## 5. 録画設定

「設定・データ管理」→「録画・動画再生設定」で録画品質を調整できます。

### 推奨設定

チャートの録画はそれほど高い品質は必要ないため、デフォルトは低品質に設定しています。

| 設定項目 | デフォルト値 | 補足 |
|---|---|---|
| フレームレート | 30fps | 通常のトレード記録に十分 |
| 録画品質 | 低品質（小容量） | チャート録画向け。ストレージ節約を優先した設定 |

> 画質を上げたい場合は「標準」以上に変更してください。品質が高いほど録画ファイルのサイズが大きくなります。

---

## 6. セットアップ完了

以上でセットアップは完了です。

次のステップ:

1. 録画を開始する（左メニュー下部の「録画開始」ボタンを押すか、「録画・動画再生設定」画面でスケジュール録画を予約する）、MT5 でトレードする
2. トレード後に CSV を取込フォルダに配置し、「CSV 取込」から取り込む
3. 「取引履歴」画面でトレードを選択すると、録画動画と照合して再生できます
