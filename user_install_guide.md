# FX Trade Insight インストール手順書

## 動作環境

| 項目 | 最低 | 推奨 |
|---|---|---|
| OS | Windows 10（64bit） | Windows 11（64bit） |
| メモリ | 8GB | 16GB 以上 |
| ストレージ | 空き 500MB（アプリ本体） | SSD 推奨（録画ファイルの保存先含む） |
| GPU | 不要 | NVIDIA / AMD GPU（録画エンコードの高速化に有効） |
| MT5 | 任意 | チャートプロット・動画再生連携を使う場合に必要 |
| ネットワーク | ライセンス認証時にインターネット接続が必要 | |

> 録画機能を使う場合、長時間の録画ファイルは数GB以上になることがあります。録画保存先には十分な空き容量のあるドライブを指定してください。

---

## 1. インストーラーのダウンロード

ご購入先に応じて以下の手順でダウンロードしてください。

**BOOTH で購入した場合**
1. BOOTH の購入履歴ページを開く
2. 「FX Trade Insight」の「ダウンロード」ボタンをクリックする
3. `FxTradeInsight-Setup-x.x.x.exe` をダウンロードする

**Note で購入した場合**
1. ご購入後に案内された Google Drive のリンクを開く
2. 「ウイルス スキャンを実行できません」という警告が表示された場合は、画面下部の「**このままダウンロード**」をクリックしてください
3. `FxTradeInsight-Setup-x.x.x.exe` をダウンロードする

---

## 2. インストールの実行

1. ダウンロードした `.exe` ファイルをダブルクリックして実行する
2. インストールウィザードの指示に従い「次へ」をクリックする
3. インストール先フォルダを確認して「インストール」をクリックする
4. 完了後、デスクトップまたはスタートメニューに「FX Trade Insight」が追加される

---

## 3. よくある警告への対処

### Windows SmartScreen の警告が出た場合

コードサイニング証明書が未取得のため、初回実行時に SmartScreen の警告が表示されます。

1. 「Windows によって PC が保護されました」という画面が表示される
2. **「詳細情報」** をクリックする
3. **「実行」** をクリックする

> この警告は署名なし exe に表示されるものであり、アプリ自体に問題はありません。

### セキュリティソフトに誤検知された場合

一部のセキュリティソフトが Electron アプリを誤検知することがあります。

**誤検知の原因となる正常な動作：**

- `ffmpeg.exe` による画面録画処理
- ローカルポート（`127.0.0.1:8765`）での HTTP 通信（MT5 との連携用）
- ライセンス認証のための HTTPS 通信（Google Apps Script）

これらはすべてアプリの正常な動作です。悪意のあるコードは含まれていません。

**対処手順：**

1. セキュリティソフトの設定を開く
2. `FxTradeInsight-Setup-x.x.x.exe` および `FX Trade Insight` をスキャン対象から除外（ホワイトリスト）に追加する
3. インストールを再実行する

**主要ソフト別の対処例：**

| ソフト | 対処方法 |
|---|---|
| Windows Defender | 「脅威の履歴」→ 対象ファイルを選択 →「許可」 |
| ESET | 「除外リスト」に実行ファイルのパスを追加 |
| Norton | 「信頼するファイル」に追加 |

ご不明な場合は **お問い合わせ** ください。

## 4. アンインストール方法

1. Windows の「設定」→「アプリ」→「インストールされているアプリ」を開く
2. 「FX Trade Insight」を選択して「アンインストール」をクリックする

> アンインストールしても取引履歴データ（SQLite）は削除されません。データも完全に削除したい場合は、アンインストール前にアプリ内の「設定・データ管理」→「全データ削除」を実行してください。

---

## 5. アップデート方法

現在のバージョンは**手動アップデート方式**です。

1. アプリ内の「About」画面でバージョンを確認する
2. 新バージョンが案内されたら Google Drive から新しい `.exe` をダウンロードする
3. 既存のアプリを上書きインストールする（データは引き継がれます）
